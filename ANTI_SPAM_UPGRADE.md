# 防广告系统升级说明 v6.0

## 升级概述

在原有防广告系统基础上，新增三大智能检测模块，大幅提升广告识别准确率。

---

## 🆕 新增功能

### 1️⃣ 行为分析模块

**功能说明：** 通过分析用户发送消息的时间模式和内容特征，识别机器人和批量发送行为。

**检测维度：**

- ⚡ **快速连续发送检测**
  - 30秒内发送5条以上消息 → +40分
  - 识别批量广告推送行为

- 🤖 **固定间隔模式检测**
  - 检测消息时间间隔的规律性
  - 间隔标准差 < 3秒 → +50分
  - 典型机器人发送特征

- 🔁 **重复内容检测**
  - 基于文本相似度算法（Jaccard）
  - 相似度 > 70% 的消息对超过2对 → +35分
  - 识别复制粘贴的广告模板

- 🆕 **新账号爆发检测**
  - 首次接触后5分钟内发送8条以上 → +30分
  - 典型广告账号行为模式

**配置参数：**
```javascript
BEHAVIOR_RAPID_MESSAGE_COUNT: 5          // 快速发送阈值
BEHAVIOR_RAPID_MESSAGE_WINDOW: 30        // 检测窗口（秒）
BEHAVIOR_INTERVAL_TOLERANCE: 3           // 间隔容差（秒）
BEHAVIOR_SIMILARITY_THRESHOLD: 0.7       // 相似度阈值
BEHAVIOR_HISTORY_MAX: 20                 // 保存历史消息数
```

---

### 2️⃣ 社交图谱分析模块

**功能说明：** 追踪和关联具有相同特征的多个账号，识别批量注册的垃圾账号群组。

**检测维度：**

- 🔗 **指纹关联检测**
  - 基于域名、用户名、电话号码、关键词组合生成指纹
  - 同一指纹下3个以上账号 → +60分
  - 自动识别广告集群

- 🚫 **封禁账号关联**
  - 检测是否与已封禁账号共享指纹
  - 关联2个以上被封账号 → +70分
  - 自动阻断账号群组

**指纹类型：**
1. **域名指纹** - 相同推广域名的账号
2. **用户名指纹** - 相同TG用户名的账号
3. **电话指纹** - 相同号码段的账号
4. **关键词指纹** - 相同垃圾关键词组合
5. **结构指纹** - 相同消息排版模式

**配置参数：**
```javascript
SOCIAL_FINGERPRINT_ACCOUNTS_THRESHOLD: 3  // 集群账号数阈值
SOCIAL_FINGERPRINT_TTL_SECONDS: 604800    // 指纹有效期 7天
SOCIAL_CLUSTER_SCORE_BOOST: 60            // 集群账号额外分数
```

**示例场景：**
- 账号A发送包含 `pandaonline.world` 的广告
- 账号B、C也发送包含该域名的消息
- 系统自动关联A、B、C为同一广告集群
- 后续账号D发送该域名时立即被高度怀疑

### 3️⃣ 智能关键词检测模块

**功能说明：** 突破传统精确匹配限制，识别各种变形、混淆的广告内容。

**检测技术：**

#### 📝 拼音/谐音检测
识别常见的同音替换和变形写法：

| 标准词 | 变形示例 |
|--------|----------|
| 微信 | 薇信、威信、维信、vx、v信、wx、w信 |
| QQ | q q、扣扣、кυ、ⓠⓠ |
| Telegram | 电报、纸飞机、飞机、tg、t g |
| 赚钱 | 挣钱、赚米、搞钱、来钱 |
| 博彩 | 柏菜、博采、搏彩 |
| 刷单 | 刷d、刷丹 |

检测到变形 → +25分/个

#### 🔤 字符变形还原
自动识别和还原混淆字符：

| 标准字符 | 变形字符 |
|----------|----------|
| a | а, ɑ, α, @, 4 |
| e | е, ė, ē, 3 |
| i | і, ι, 1, l, \| |
| o | о, ο, 0, ø |
| s | ѕ, ś, $, 5 |

**示例：**
- `vх` (西里尔字母x) → `vx`
- `微іn` → `微in`

#### 🔍 增强正则检测
识别各种混淆手法：

1. **电话混淆** - `1 3 8-1 2 3 4-5 6 7 8` → +25分
2. **域名混淆** - `w w w . example . com` → +30分
3. **TG链接混淆** - `t . me / username` → +35分
4. **微信号模式** - `wx123456`, `vx:abc123` → +30分
5. **QQ号模式** - `qq:123456789` → +28分
6. **广告排版** - `👉👉👉 内容 🔥🔥🔥` → +25分
7. **价格诱导** - `仅需$99`, `低至¥9.9` → +20分
8. **紧急促销** - `限时优惠`, `仅剩3个名额` → +22分

#### 😀 Emoji隐写检测
识别用emoji编码信息的隐蔽手法：
- emoji数量 ≥ 5
- 有规律的重复模式
- 检测到 → +30分

## 📊 综合评分系统

**风险等级：**
- **0-59分** - 正常，直接通过
- **60-109分** - 可疑，进入人工审核
- **110-179分** - 高风险，直接拦截
- **180+分** - 极高风险，自动封禁

**多维度叠加：**
```
总分 = 内容评分 + 行为评分 + 社交图谱评分
```

**示例：**
```
基础内容: 45分 (链接+联系方式)
+ 行为分析: 50分 (固定间隔发送)
+ 社交图谱: 60分 (属于已知广告集群)
+ 变形关键词: 50分 (检测到"薇信"、"вх")
------------------------------------------
总分: 205分 → 自动封禁
```

## 📚 技术细节

### 文本相似度算法
使用 Jaccard 相似度计算（基于字符 bigram）：
```
相似度 = |A ∩ B| / |A ∪ B|
```

### 指纹生成策略
```
指纹 = SHA256(特征类型:标准化特征值)
```

### 行为模式分析
```
标准差 = √(Σ(x - μ)² / n)
规律性 = 标准差 < 阈值
```

## ⚠️ 注意事项

1. **误杀风险**
   - 新系统更严格，可能增加误判
   - 建议初期密切关注审核队列
   - 及时使用 `/trust` 信任正常用户

2. **性能影响**
   - 每条消息处理时间略有增加
   - 不影响整体并发能力
   - KV 读写频率提高

3. **隐私考虑**
   - 行为数据保存时间有限（5分钟窗口）
   - 社交图谱仅存储用户ID，不存储消息内容
   - 自动过期清理

4. **持续优化**
   - 建议定期检查 `/cleanup` 清理无效数据
   - 观察日志中的误判模式
   - 根据实际情况调整参数

## 📞 问题反馈

如遇到以下情况请及时调整：

- ❌ 正常用户被频繁拦截 → 提高阈值或关闭部分检测
